HubSpot and Caldera Forms Integration

== Description ==

The Integration of HubSpot and Caldera Forms plugin lets you seamlessly connect and sync data with HubSpot CRM. The plugin simply adds a new HubSpot Processor to Caldera form. When the Form is filled the plugin automatically syncs data from your Caldera form to your HubSpot CRM.

Using this plugin you can easily connect the two powerful tools and grow your leads. 


=== FEATURES ===

- Adds New HubSpot Processor in the Caldera Forms
- Easily Integrate with HubSpot using API Key (Learn how to get your API Key and use this plugin)
- Sandbox and Production mode options are available
- Collect First Name, Last Name and Email Address details
- Easily collect data and push to HubSpot
 

== PRO VERSION ==

- All the features of the free version
- Sandbox and Production mode options available
- Dynamically fetch field list from HubSpot
- Add selected form fields for merge tags to your HubSpot Contact list
- Can add many Custom Fields

== Installation ==
1. Login to your WordPress dashboard and navigate to Plugins > Add New
2. Search for "Integration of HubSpot and Caldera Forms".
3. Click install.
4. Click activate.
